windspharm

windspharm is a Python package I created for doing computations on global vector fields in spherical geometry. It provides a high level interface for computations involving spherical harmonic transformations. The package supports fields with metadata using CDAT or Iris. The package is also bundled with the UV-CDAT distribution.
